# 图片导出与压缩插件 隐私政策

最后更新：2026-03-24

感谢你使用「图片导出与压缩」Figma 插件。我们重视并尊重你的隐私。本隐私政策用于说明本插件会处理哪些数据，以及如何处理这些数据。

## 1. 我们处理的数据

本插件仅在实现功能所必需的范围内处理数据：

- 你在插件中输入的 TinyPNG API Key（用于图片压缩）
- 你在当前 Figma 文件中主动选择并导出的图层内容（用于导出与可选压缩）
- 基本功能配置（如是否已展示提示、压缩计数等）

## 2. 数据的使用方式

- TinyPNG API Key 仅用于向 TinyPNG 服务发起压缩请求鉴权。
- 导出图片仅在你主动执行导出/压缩操作时被处理。
- 当你启用压缩功能时，图片会发送到 TinyPNG 服务进行压缩。

## 3. 本地存储

插件会通过 Figma 提供的本地存储能力（clientStorage）保存必要配置，例如：

- TinyPNG API Key
- 是否已展示过引导提示
- 压缩次数计数

这些数据用于提升你的使用体验，不用于广告或画像。

## 4. 第三方服务

本插件在你使用相关功能时可能访问以下第三方服务：

- TinyPNG API（图片压缩）
- 代理服务（用于网络请求转发）

我们不会将你的数据出售给第三方。

## 5. 我们不做什么

- 不会收集与你身份直接关联的个人信息（如姓名、手机号等）
- 不会在自有服务器长期存储你的设计文件或导出图片
- 不会将你的数据用于营销目的

## 6. 数据安全

我们采取合理措施减少不必要的数据处理范围，仅处理实现功能所需的最小数据集。

## 7. 你的选择

- 你可以随时停止使用插件
- 你可以在插件内更新或清空 API Key（如适用）
- 你可以在 Figma 侧管理插件授权与使用

## 8. 政策更新

我们可能会根据功能更新或合规要求调整本政策。更新后会在本页面发布最新版本。
